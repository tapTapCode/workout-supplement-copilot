Step 1: Get Image IDs
  - Go to Media Library
  - Click on image
  - Check URL: wp-admin/upload.php?item=123
  - The number (123) is the ID

Step 2: Use Shortcode
  [product_gallery ids="1,2,3,4,5" columns="4" size="medium"]

Shortcode Parameters:
  ids="1,2,3"          - Image IDs (comma-separated, REQUIRED)
  columns="4"          - Number of columns (2-5, default: 4)
  size="medium"        - Image size (thumbnail/medium/large/full)
  show_title="yes"     - Show titles (yes/no, default: yes)
  show_description="no" - Show descriptions (yes/no, default: no)
  layout="grid"        - Layout type (grid/masonry, default: grid)
