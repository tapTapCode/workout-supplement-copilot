# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-11-07

### Added
- Initial release of ShopLaz Reviews plugin
- Modern review display with star breakdown and horizontal testimonial cards
- Multi-platform integration (WordPress, Shopee, Lazada)
- Store URL management admin page
- Automatic review sync (hourly scheduled)
- Shopee Open Platform API integration
- Lazada Open Platform API integration
- REST API endpoints for review management
- WooCommerce integration
- Shortcode support `[shoplaz_reviews]`
- AJAX review submission
- Review approval workflow
- Admin dashboard for review management
- Settings page for plugin configuration
- Database schema for reviews and store management

### Features
- Star rating system (1-5 stars)
- Review breakdown visualization
- Horizontal scrollable testimonial cards
- Profile pictures and reviewer information
- Review deduplication by external ID
- Support for multiple regions (SG, MY, PH, TH, VN, ID, TW)
