# Course Transcripts

Add your course content here. Each course should be a separate `.md` file.

## File Format

Name your files descriptively:
- `natural-hair-101.md`
- `pricing-masterclass.md`
- `client-retention-secrets.md`

## Template

Use this structure for each course file:

```markdown
---
title: "Course Title Here"
category: "hair_education" or "business_mentorship"
description: "Brief description of what this course teaches"
---

# Module 1: Title

[Your transcript or detailed notes here]

Include:
- Key concepts you teach
- Examples you use
- Stories you share
- Step-by-step instructions

# Module 2: Title

[Continue with more content]

---

## Key Takeaways

- Main point 1
- Main point 2
- Main point 3
```

## Tips for Best Results

1. **Include your examples** - The specific stories and examples you use
2. **Keep your voice** - Write how you actually talk/teach
3. **Add context** - Explain WHY, not just WHAT
4. **Include Q&A** - Common questions students ask during this content
